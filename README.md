# The C Programming Language — Study Archive

A structured study archive for **The C Programming Language, 2nd Edition** by Brian W. Kernighan and Dennis M. Ritchie.

## Goals

- Understand C language fundamentals through implementation
- Master standard I/O, pointers, structures, and system interfaces
- Build reliable, portable C programs
- Document practical insights and edge cases

## Structure

```
src/
├── ch01_tutorial_introduction/
├── ch02_types_operators_expressions/
├── ch03_control_flow/
├── ch04_functions_and_program_structure/
├── ch05_pointers_and_arrays/
├── ch06_structures/
├── ch07_input_and_output/
└── ch08_unix_system_interface/
```

Each chapter contains:
- `examples/` — independently rewritten sample code
- `exercises/` — personal exercise solutions

## Naming Conventions

| Type | Pattern |
|------|---------|
| Examples | `ex_<chapter>_<topic>` |
| Exercises | `exercise_<chapter>_<number>` |

## Environment

- IDE: Visual Studio 2022
- Language: C (C99 or later)
- Platform: Windows / POSIX
