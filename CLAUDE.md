# CLAUDE.md

## Repository Overview

This repository contains a structured study archive for **The C Programming Language, 2nd Edition** by Brian W. Kernighan and Dennis M. Ritchie.

It focuses on:
- core C language fundamentals
- practical implementations of standard algorithms and data structures
- exercise solutions and experimentation
- building a reusable reference for C programming

## Purpose

The goals of this repository are to:

- understand C language fundamentals through implementation
- master standard I/O, pointers, structures, and system interfaces
- build reliable, portable C programs
- document practical insights and edge cases
- create a long-term engineering reference

## Structure

```text
src/
├─ ch01_tutorial_introduction/
├─ ch02_types_operators_expressions/
├─ ch03_control_flow/
├─ ch04_functions_and_program_structure/
├─ ch05_pointers_and_arrays/
├─ ch06_structures/
├─ ch07_input_and_output/
└─ ch08_unix_system_interface/
```

Each chapter contains:
- `examples/` (rewritten sample code)
- `exercises/` (personal solutions)
- optional `notes.md`

## Naming Conventions

- examples: `ex_<chapter>_<topic>`
- exercises: `exercise_<chapter>_<number>`

## Development Environment

- IDE: Visual Studio 2022
- Language: C
- Platform: Windows / POSIX
- Standard: C99 or later

## Study Approach

1. Understand the concept
2. Rebuild sample code independently
3. Test behavior with simple cases
4. Solve exercises without copying
5. Document findings briefly

## Study Rules

1. Do not copy the book verbatim
2. Focus on understanding language behavior
3. Keep code simple and explicit
4. Separate examples and exercises
5. Record mistakes and edge cases
